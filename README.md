# Capstone 3: EmporiUm Northeast Region Sales Analysis
10 Minute presentation link : https://drive.google.com/file/d/1NJJs23Hyedb-_d6f7PgbDNmXm0MBxZbd/view?usp=sharing
## Project Overview
A Power BI report analyzing four years of sales data (2022-2025) for EmporiUm, 
a virtual student bookstore, across the Northeast region (Maryland, Massachusetts, 
New Jersey, and Maine).

## Data Sources
- Capstone3_Sample_Sales.xlsx (store sales, online sales, products, store locations)
- book_list.txt (general audience book titles and authors)

## Report Pages
1. Sales Overview — store and online trends, category performance
2. Regional Breakdown — state share, revenue per store
3. State Performance — year over year by state, summary table, waterfall chart
4. Store Performance — top 10 stores, North Harford analysis, bottom 5 stores
5. Category Trends — category shifts over time, textbook decline
6. Online Channel — online revenue by state, top products, growth trend
7. Books Performance — top general books with authors, fiction vs non-fiction
